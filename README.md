# Libraries

## angular-cli-ghpages

    url: https://www.npmjs.com/package/angular-cli-ghpages

### Commands

To install

    npm i angular-cli-ghpages --save-dev

To build

    ng build --prod --base-href "/"

To publish

    npx ngh

    ngh --dir dist/learn --no-silent --repo=git@github.com:nazmuldipu/learn.git
